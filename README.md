# Setup instructions and details
- Use the following version of Node: 10.8.1
- Run `npm install`
- Run `npm start`
- cd src/app/home and run `npx json-server card.json`
- Run `ng serve`

# JSON Server CLI
Run `npx json-server card.json` to pass card.json to JSON Server CLI
